# Was ist ein Mockup
Ein Mockup, ist eine visuelle Darstellung, die einem Prototyp ähnelt und das endgültige Aussehen und die Wirkung eines Designs simuliert. Es geht über Wireframes und Layouts hinaus, indem es die tatsächlichen Designelemente wie Farben, Typografie und Bildmaterial zeigt. Mockups bieten eine realistische Vorschau darauf, wie ein Produkt, eine Website oder eine Anwendung für die Nutzer aussehen wird
