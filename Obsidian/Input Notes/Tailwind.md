**Lern Leitfaden:**
1. Core Utility Klassen meistern (80% der Arbeit in einem Projekt)
	1. Layout
		`flex`, `flex-col`, `items-center`, `justify-between`
		`grid`, `grid-cols-2`, `gap-4`
		`container`, `mx-auto`, `w-full`, `max-w-screen-lg`
	2. Spacing
		`p-4`, `px-6`, `m-2`, `space-y-4`
	3. Typography
		`text-sm`, `text-xl`, `font-bold`, `leading-tight`
	4. Farben
		`bg-gray-800`, `text-white`, `border-gray-300`, `hover:bg-blue-600`
	5. Positioning 
		`absolute`, `relative`, `top-0`, `right-0`
2. Responsive Design
3. Komponenten wiederverwendbar machen (@apply)
4. Dark Mode
5. Design System / Farben erweitern
6. Praktische Tailwind Patterns
	1. Card Layout
		`rounded-xl shadow-lg p-6 bg-white`
	2. Flex Center
		`flex items-center justify-center`
	3. Button Base
		`inline-flex items-center px-4 py-2 rounded-lg font-medium`
	4. Hover + Transition
		`transition duration-300 ease-in-out`
