# Komponenten in React

#### Wichtige Aspekte zu Komponenten
1. React Applikationen bestehen komplett aus Komponenten
2. Es sind Bausteine in der UI von React
3. UI-Element, das über eigene Daten, Logik und Erscheinungsbild verfügt (wie es funktioniert und aussieht)
4. Wir erstellen komplexe Benutzeroberflächen, indem wir mehrere Komponenten erstellen und diese miteinander kombinieren.
5. Komponenten können wiederverwendet, ineinander verschachtelt und Daten zwischen ihnen ausgetauscht werden.
