Regressionstest sind eine Art von Softwaretests, bei denen **bestehende, bereits erfolgreich durchgeführte Testfälle wiederholt** werden.

Der Hauptzweck ist es sicherzustellen, dass neue Codeänderungen (wie Fehlerbehebungen neue Funktion oder Verbesserungen) keine unabsichtlichen negativen Auswirkungen auf bereits funktionierende Teile der Software haben.

**Beispiel:**
Es wird ein Test durchgeführt, um zu prüfen ob das Programm korrekt läuft. Später wird eine neue Funktion implementiert und danach wird der Test nochmal durchgeführt um sicherzustellen das, dass Programm weiterhin korrekt läuft.

Das ist ein Regressionstest
